## What is this?

DermAI Detector is a full-stack deep learning application that classifies skin lesions from dermoscopic photos. Upload a photo, click on the lesion вҖ” and in ~5 seconds you get:

- **Diagnosis** with confidence score
- **Grad-CAM heatmap** showing where the model looked
- **Plain-language explanation** generated by GPT-4o-mini
- **Risk level** (High / Medium / Low)

Built as a bachelor's thesis project in Biomedical Engineering.

## Installation

### Requirements

- Python 3.10+
- CUDA-capable GPU recommended (works on CPU too, slower)

### 1. Clone

```bash
git clone https://github.com/YOUR_USERNAME/skinCancer.git
cd skinCancer
```

### 2. Install dependencies

```bash
# With GPU (CUDA 11.8)
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118

# CPU only
pip install torch torchvision

# Rest of dependencies
pip install timm gradio openai opencv-python Pillow \
            scikit-learn numpy pandas python-dotenv tqdm
```

### 3. API key (optional)

Create `.env` in project root:

```env
OPENAI_API_KEY=sk-your-key-here
```

Without it вҖ” GPT explanations are disabled, everything else works fine.

### 4. Get model weights

Weights are not stored in this repo.  
Download `melanoma_best_v3.pth` and `skin_detector_v2.pth` from the [Releases](../../releases) page and place them in `skinCancer/`.

### 5. Run

```bash
cd model_test.py
python app.py
```

Open **http://localhost:7860** in your browser.

## Training from scratch

### Dataset

Download HAM10000 from Kaggle (~3.5 GB):

```
https://www.kaggle.com/datasets/kmader/skin-lesion-analysis-toward-melanoma-detection
```

Place images in:
```
skinCancer/dataset/archive/all_images/   # 10015 Г— .jpg
skinCancer/dataset/archive/HAM10000_metadata.csv
```

### Train lesion classifier

```bash
cd model_training
python model_training.py
```

Config: EfficientNet-B4 В· 15 epochs В· AdamW lr=1e-4 В· CosineAnnealingLR  
Time: ~4.5h on RTX 4060

### Train skin detector

```bash
python train_skin_detector.py
```

Config: MobileNetV3-Small В· 12 epochs В· AdamW lr=3e-4 В· ReduceLROnPlateau
